# Basic Gulp configuration for fast Web Development
## Installation:
` npm i gulp gulp-cssnano gulp-autoprefixer gulp-sass browser-sync gulp-rename -D `
## What you gives?
* SASS compiling
* SASS injecting
* Renaming
* Autoreload with Browsersync
* Autoprefixing CSS
* Minifying CSS with cssnano plugin
